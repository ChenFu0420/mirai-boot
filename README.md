# Mirai-boot

基于[Go-Mirai-Client](https://github.com/ProtobufBot/Go-Mirai-Client)
与[pbbot-spring-boot-starter](https://github.com/ProtobufBot/pbbot-spring-boot-starter)
开发的QQBot.

## 环境需求
1. Redis
2. MySQL
3. Nacos
4. JDK11

### 已功能实现

1. 色图搜索
2. 原神体力计算器
3. 新番更新提醒
4. Telegram群与QQ群消息互通
